1. ¿Que es una Gaceta?

La gaceta era en su origen un periodico veneciano que se publicaba
una vez a la semana a principios del siglo XVII. De este modo
Se entendio por "gaceta" al periodico de política, noticias, teatros, moda
tribunales o solamente alguna de estas materias

2. ¿Para que sirve el Html?

El lenguaje html (hypertext markup language) se utiliza para el desarrollo y creacion de paginas web.
Se compone de una serie de etiquetas que el navegador interpreta.
Entre las etiquetas que se incluyen dentro del lenguaje HTML se encuentran: hipervinculos,
etiquetas para imagenes, saltos de pagina, entre otras
